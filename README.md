# 💄 LUXEA Beauty Marketplace

เว็บไซต์ขายเครื่องสำอางออนไลน์ สำหรับโปรเจกต์การศึกษา

## 🚀 Features

- ✅ การลงทะเบียนและเข้าสู่ระบบ (Customer / Admin)
- ✅ ค้นหาและเรียกดูสินค้า
- ✅ ตะกร้าสินค้า
- ✅ คำนวณค่าจัดส่ง (ฟรีเมื่อซื้อครบ 699 บาท)
- ✅ ชำระเงิน
- ✅ ระบบจัดการสินค้า (Admin)
- ✅ คะแนนสะสม (Loyalty Points)
- ✅ สถานะออเดอร์

## 🛠 Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: In-Memory (for demo)
- **Tools**: Git, GitHub, SourceTree

## � Installation & Run

1. Clone repository
2. Go to `beauty-studio` directory
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run development server:
   ```bash
   npm run dev
   ```
5. Open browser: http://localhost:3000
